Active Headphones for Selective Source Isolation

Project Overview

Custom-built headphones with enabled voice signature selectivity for predictively tracking dynamic voices to promote attention and focused learning in distracting environments.

Key Contributions:

  • Collaborated with a team of four to manage project scope and deliverables.
  • Implemented efficient signal sampling algorithms and real-time attentive machine learning inference.
  • Achieved hybrid-device signal tracking and active background suppression with user-interface signal selection.
