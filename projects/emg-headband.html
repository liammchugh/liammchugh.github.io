Project Overview

Developed a wireless headband electromyographic (EMG) device for neuro-controlled multidimensional guitar pedal effects modulation.

Key Contributions:

  • Developed ICA software and microcomputer system architecture for real-time signal processing.
  • Implemented neuro-controlled modulation for multidimensional effects on guitar pedals.
  • Wireless functionality and real-time signal analysis were achieved.
